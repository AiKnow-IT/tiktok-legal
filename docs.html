High-level overview of the TikTok integration.

TikTok end-to-end flow

  1. OAuth (Login Kit): user grants access.
  2. Fetch creator settings: creator_info/query (privacy options).
  3. Init publish: /v2/post/publish/video/init.
  4. Deliver video: FILE_UPLOAD (upload_url) or PULL_FROM_URL (public MP4 URL).
  5. Check result: status/fetch until completed.

Scopes

  • video.publish — required to publish videos.

Data handling

  • We store OAuth refresh tokens securely to refresh access tokens.
  • We store minimal metadata (caption/privacy) and limited logs for debugging.
