Support

Need help with the TikTok integration or publishing workflow? Contact us below.

Contact

Email: business.ai.gpt@gmail.com

We typically respond within 2 business days.

What to include in your support request

  • Short description of the issue (what you expected vs. what happened)
  • Date/time of the workflow run
  • Any error message or status code
  • Whether you are using FILE_UPLOAD or PULL_FROM_URL

Please do not send access tokens or secrets via email.

Common topics

  • OAuth / login issues: make sure the redirect URL matches exactly and scopes are enabled.
  • Publishing fails: check creator privacy options and posting status fetch results.
  • URL verification: ensure the required verification file is publicly accessible on your website.
