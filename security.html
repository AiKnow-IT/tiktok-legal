Security

Security & Data Protection

How we protect your accounts, data, and privacy.

Security Overview

AIKnow-IT is built with security as a core principle. We handle sensitive data — including social media OAuth tokens — with the highest level of care. This page outlines our security practices and how we protect your information.

Authentication & Authorization

OAuth 2.0

All social media connections use the official OAuth 2.0 authorization flows provided by each platform (TikTok, YouTube, Instagram). This means:

  • We never see or store your social media passwords.
  • You authorize AIKnow-IT directly through the platform's consent screen.
  • We only request the minimum required permissions (e.g., video.publish for TikTok).
  • You can revoke access at any time from your social media account settings.

Token Security

  • OAuth access tokens and refresh tokens are encrypted at rest using industry-standard encryption.
  • Tokens are never exposed in URLs, logs, or client-side code.
  • Token refresh is handled server-side with secure credential management.
  • Expired or revoked tokens are immediately purged from our systems.

Data Protection

Encryption

  • In transit: All communications between your browser, our servers, and third-party APIs use TLS 1.2 or higher.
  • At rest: Sensitive data (OAuth tokens, account credentials) is encrypted using AES-256 encryption.

Data Minimization

We follow the principle of data minimization:

  • We only collect data that is strictly necessary for the Service to function.
  • We only request the minimum OAuth scopes needed for publishing.
  • Generated videos are stored temporarily and automatically cleaned up after publishing.
  • No unnecessary personal data is collected or retained.

Access Controls

  • Access to user data and OAuth tokens is strictly limited on a need-to-know basis.
  • All administrative access requires multi-factor authentication.
  • We maintain audit logs of all access to sensitive data.

Infrastructure Security

  • The platform is hosted on secure, reputable cloud infrastructure with built-in DDoS protection.
  • All API endpoints are rate-limited to prevent abuse.
  • Regular security updates and patches are applied to all systems.
  • Database backups are encrypted and stored in geographically separate locations.

Third-Party Security

We carefully evaluate all third-party services we integrate with:

  • OpenAI: Used for AI content generation. No personal user data is sent to OpenAI — only content generation prompts.
  • TikTok, YouTube, Instagram: Official APIs used with minimum required permissions.
  • Supabase: Cloud storage for temporary video hosting with encryption and access controls.

Incident Response

In the event of a security incident:

  • We will investigate and contain the incident as quickly as possible.
  • Affected users will be notified within 72 hours, in compliance with GDPR requirements.
  • We will take immediate steps to mitigate any potential harm.
  • A post-incident review will be conducted to prevent recurrence.

Responsible Disclosure

If you discover a security vulnerability in AIKnow-IT, please report it responsibly to business.ai.gpt@gmail.com with the subject line "Security Vulnerability Report". We will acknowledge your report within 48 hours and work to resolve the issue promptly.

GDPR Compliance

AIKnow-IT is operated from the Netherlands and fully complies with the EU General Data Protection Regulation (GDPR). For full details on how we handle personal data, please see our Privacy Policy.
