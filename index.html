What we do

AIKnow-IT helps users create short, kid-friendly animated videos (age 8–12) and publish them to social platforms. Users connect their own accounts and control what gets posted.

How it works

  1. Generate a kid-safe script and storyboard.
  2. Create a vertical short video (9:16, ~20–25 seconds).
  3. Prepare captions and hashtags.
  4. User connects TikTok via OAuth consent.
  5. Publish and track status automatically.

TikTok integration

  • Login (OAuth): Users authorize access to their TikTok account.
  • Scope: We request video.publish only for posting videos.
  • Content Posting API: We publish via Direct Post and fetch status until complete.
  • Privacy: We query creator settings and use allowed privacy options.

Kids-safety

  • No adult content, violence, hate, or harassment.
  • Positive themes: friendship, learning, humor, curiosity.
  • Content generation follows strict guidelines before publishing.
