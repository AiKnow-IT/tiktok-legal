How it works

What we do

AIKnow-IT helps creators and educational publishers generate short, kid-friendly animated videos and publish them to social platforms. Users connect their own accounts and control what gets posted.

How it works

  1. Generate a kid-safe script and storyboard (age 8–12).
  2. Create a short video (20–25 seconds, 9:16).
  3. Prepare captions and hashtags.
  4. Connect TikTok via OAuth (user consent).
  5. Publish via TikTok Content Posting API and track status.

Kids-safety

  • No adult content, violence, hate, or harassment.
  • Positive themes: friendship, learning, humor, curiosity.
  • Content is generated using strict guidelines before publishing.
